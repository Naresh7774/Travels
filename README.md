# Naresh Travels

A responsive website for exploring Indian tourist destinations and travel packages.

## Features

- Home page with full-screen hero section
- Famous Places - showcasing 6 popular Indian destinations
- Travel Details - travel information and tips
- Hotels and Packages pages
- Contact page
- Responsive design (mobile, tablet, desktop)
- Sticky navigation bar

## Pages

- **index.html** - Home with hero section
- **famous-places.html** - Popular destinations
- **travel-details.html** - Travel information, video, and maps
- **packages.html** - Travel packages
- **hotels.html** - Hotel listings
- **contact.html** - Contact form
- **css/style.css** - Styling

## Tech Stack

- HTML5
- CSS3
- Font Awesome Icons
- Google Maps Embed
- YouTube Embed
- Unsplash Images

## How to Use

1. Open any `.html` file in a browser
2. Navigate between pages using the top menu
3. All content is responsive and works on mobile
